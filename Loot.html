Mission Events

Machine Event

The Machine events are able to be spotted when a Core infuser is found near a large piece of machinery. Activating the machine causes to deal with whatever event is thrown at you. Once completed you are able to use a Blank matrix core in the infuser to get a randomly generated Overclock for the class you choose.

Cargo Crate

The Cargo Crate is found randomly throughout the cave. The Crates are missing two batterys that can be found nearby. Putting both batterys back into the crate allows the Dwarves to recieve minerals and a random weapon framework or paint job.

Lost pack

The lost pack can be found by finding a Blinking lost helmet stuck in the ground, Interacting with the helemt updates the map showing a purple cross on the map. Digging into this area grants the Dwarves minerals and a random Pickaxe paintjob or framework piece.
